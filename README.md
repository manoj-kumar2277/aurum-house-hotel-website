## 📖 Project Overview

**Aurum House** is a fictional boutique hotel brand, designed as a complete front-end case study covering discovery, information, browsing, and booking. The site is built entirely with hand-written HTML5, CSS3, and vanilla JavaScript — no frameworks, no build tooling, just clean static files ready to deploy anywhere.

The visual identity pairs a warm ivory and cream base with deep burgundy and olive-green accents, serif display type for headings, and a restrained, editorial layout that feels closer to a print brochure than a typical template.

## ✨ Features

### Frontend Design
- Consistent color theme, typography, and spacing system across all pages
- Sticky navigation with active-page indication
- Card-based layouts for rooms, amenities, and gallery items
- Custom-styled buttons, forms, and footer
- Editorial, gallery-style visual language (line-art motifs alongside photography)

### Responsive Design
- Fully responsive across mobile, tablet, and desktop
- Built with CSS Grid, Flexbox, and breakpoint-based media queries
- Collapsible mobile navigation drawer

### JavaScript Interactivity
- Mobile menu toggle
- Sticky header on scroll
- FAQ accordion (Rooms page)
- Gallery lightbox viewer
- Booking form validation with inline error messages
- Interactive 3D hero animation (Three.js), with `prefers-reduced-motion` support for accessibility

### SEO
- Unique `<title>` and meta description on every page
- Meta keywords per page
- One `<h1>` per page with a clear `<h2>` / `<h3>` heading hierarchy
- Descriptive internal links between pages
- Meaningful `alt` text on every image

## 🚀 Deployment

1. Push the project to a public GitHub repository.
2. In Netlify: **Add new site → Import an existing project → Deploy with GitHub**, then select the repository.
3. No build command is required — this is a static site, deployed as-is.
4. Netlify generates a live `.netlify.app` URL, added to the [Live Links](#-live-links) table above.
